# 동행복권 로또 6/45 통계 시각화 사이트 PRD

## 제품 개요
- 동행복권 로또 6/45의 1회부터 최신 회차까지 당첨번호를 수집한다.
- 번호별 출현 패턴을 통계량과 시각화 중심으로 탐색할 수 있게 한다.
- 회차 순 조회, 번호별 정렬, 인사이트 카드, 조건 기반 추천 조합, 모의 구매 기능을 제공한다.
- 매주 토요일 오전 9시(KST)에 최신 회차 데이터를 자동 반영한다.

## 핵심 사용자
- 일반 공개 사용자
- 수학 수업에서 확률과 통계를 설명하려는 교사와 학생
- 숫자 분포와 패턴을 직관적으로 탐색하려는 사용자

## 핵심 기능
- 번호별 전체 출현 횟수 막대그래프
- 번호순, 출현 횟수순, 미출현 연속 회수, 최대 연속 출현 등 다양한 정렬
- 회차 순 로또번호 조회 및 회차 상세 화면
- 최근 추세, 안정성, 변동성, 보너스 편향 등 확장 통계
- 1등 당첨금, 당첨자수, 자동·수동·반자동 비중 통계
- 조건 기반 추천 번호 생성기
- 모의 로또 구매와 다음 회차 자동 채점

## 데이터 정책
- 우선순위 1: 동행복권 공식 JSON 엔드포인트 시도
- 우선순위 2: 공식 페이지 파싱 시도
- 우선순위 3: 공개 백업 데이터셋 사용
- 보너스 번호는 통계에 포함한다.

## 배포 및 운영
- 프론트엔드: Next.js App Router
- 배포: Vercel
- 자동 업데이트: GitHub Actions 스케줄러
- 스케줄: 매주 토요일 09:00 KST = 토요일 00:00 UTC

## 유의사항
- 통계와 추천 조합은 관찰용 자료이며 당첨 확률 향상을 보장하지 않는다.
- 교육용 활용을 고려해 해석은 중립적이고 통계 중심으로 유지한다.
